# 🎮 Tic Tac Toe Game (Python + Tkinter)

A simple **Tic Tac Toe GUI game** built using **Python** and **Tkinter**.
This project allows two players to play Tic Tac Toe with a clean graphical interface.

---

## 🚀 Features

* ✅ 2 Player Game (X vs O)
* ✅ GUI using Tkinter
* ✅ Automatic Winner Detection
* ✅ Turn Indicator
* ✅ Winning Highlight
* ✅ Simple & Clean UI

## 🎯 Game Rules

* Player **X** starts first
* Players take turns
* First player to align 3 symbols wins
* Game highlights winning combination
